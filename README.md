# NOTEBOOK + TONIC

## Web version 

Website is built with [Vite](https://vitejs.dev/guide/) using [Tonic](https://tonicframework.dev/) - a low profile component framework for the web.

To run in a browser: 
<br>
run vite first:
```sh
npm run dev
```
then open local server:
```sh
http://localhost:5173/
```
